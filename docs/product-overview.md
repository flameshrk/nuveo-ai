Nuveo is an AI-powered Chrome Extension that analyzes resumes and generates structured, ATS-optimized feedback in seconds.

This document explains the product goals, value proposition, and user experience.


## Product Goal

Help job applicants instantly improve their resumes using AI, without needing:
- a career coach  
- long feedback cycles  
- expensive consulting services  

Nuveo provides **clear, actionable, section-by-section improvements**.


## Key Features

- Upload a PDF directly inside the Chrome extension  
- Instant AI analysis based on GPT models  
- Structured feedback:
  - Overall suggestions
  - Summary rewrite
  - Bullet-by-bullet improvements
  - ATS keyword optimization
- Clean, modern UI
- Privacy-first: no file storage  
- Export results to PDF (production)


## Why It Works

Nuveo uses:
- A curated industry knowledge base  
- Prompt engineering  
- GPT large language models  
- Resume best practices  
- ATS keyword patterns  

Together, this produces feedback that is:
- actionable  
- specific  
- non-generic  
- aligned with modern hiring standards  


## Target Users

- Students  
- Junior employees  
- Mid-level professionals  
- Job seekers applying to global companies  


## Business Model (Production Version)

- Paywall via Gumroad Licensing  
- One-time unlock  
- Future potential:
  - Subscription model
  - API for universities
  - HR partners
  - Analytics dashboard  

Nuveo is designed as a lightweight, easy-to-scale AI tool.
